# 🚗 Luxury Cars Website

A modern and visually engaging website that showcases different categories of luxury cars with an elegant design and smooth user experience.

---

##  Overview
This project presents a collection of luxury cars including **Classic, Modern, SUV, and Sports** categories.  
It is designed to highlight aesthetics, performance, and user-friendly navigation.

The website also includes a simple login system and interactive UI elements to enhance the user experience.

---

## Features
- Simple Login System (Front-end)
-  Dynamic Header with Visual Background (GIF/Video)
-  Smooth Navigation Bar
- Car Gallery with Cards Design
-  Responsive Design (Mobile Friendly)
-  Hover Effects & Animations
-  Multiple Pages Structure
